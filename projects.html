EASEPAPR
Function: low-cost, open-source Powered Air Purifying Respirator (PAPR) device that may be used as personal protective equipment (PPE) in the event that other forms of conventional PPE are unavailable.
Details:
• This PAPR alternative may be constructed largely using off-the-shelf components and used to provide breathing protection to a higher degree than improvised face masks.
• The design focuses on the powered filtration unit (blower) component of a full PAPR system.
• Future work for this project includes streamlining the design for prototyping, conducting laboratory filtration and CO​2​ testing, and integrating the device with a PAPR hood system for full-system testing.
Skills used: Autodesk Fusion 360, Markdown, Git
Created: March 2020
Type: Group - Personal

UAS Software Control Interface - Rainforest XPRIZE
Function: allows user to control their Parrot ANAFI (seen to the left) at a distance with first-person view (FPV) as part of the Rainforest XPRIZE competition.
Details:
• The Olympe API from Parrot allows our interface to communicate with the user's ANAFI.
• For safety, arcade-style controls limit the motion of the drone to the direction that its camera is facing along with turning left and right.
• Affordances indicate when each function is available to the user in flight.
• All controller functions were tested on a linux-based environment set up by me.
• The environment included a native installation of Ubuntu 18.04 LTS along with Sphinx, a virtual flight environment based on Gazebo.
Skills used: Tkinter, Python, Olympe, Sphinx, PDraw, Gazebo, Linux, Ubuntu
Created: November 2020
Type: Group - independent study [ECE391]

Timeturner
Function: Smart activity tracker and companion app to address college student work-life balance concerns amid the pandemic
Details:
• Features a dodecahedron, a 12-sided device wherein each face represents an activity set by the user.
• When the user flips the device, the activity assigned to the face that is facing up will be logged by the companion application as the current activity that is in progress. The activity that was already being logged when the flip occurred will be logged as having ended.
• Users can connect the app to their digital calendars to allow Timeturner to automatically find time in their schedule to schedule break times.
• The companion app provides insights to users about how well they are managing their time relative to their schedules set in digital calendar services.
• Both the project and presentations ("Share Outs") were selected by our professor to be used as examples of exemplary work in future semesters
Skills used: UWP, C#, Figma, Adobe Premiere Pro
Created: November 2020
Type: Group - class [ECE590: Human-Centered Computing]

GuideDog
Best Inequality Track Hack - HackDuke 2020
Wolfram Award - HackDuke 2020
Function: AI based narration camera for the visually impaired.
Details:
• Uses a pre-trained model, YoloV3, to identify objects and people in an image taken every few seconds by the user's smartphone camera.
• Google's text-to-speech service is used to generate narration to describe what the device is seeing.
• Although our vision is to make the application designed for mobile devices, the version seen in the video demo in the repo connects the user's smartphone to their desktop or laptop as an IP webcam, allowing that computer to analyze each image.
• A future release would use a webserver to apply the model to each image captured by the user's smartphone.
Skills used: Python, OpenCV, Tkinter, YoloV3, GTTS
Created: December 2020
Type: Group - Hackathon [HackDuke 2020]
